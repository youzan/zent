## Icon 图标

语义化的图标库。

### 使用指南

- 部分图标分为实心和描线两个版本, 使用 `-o`（字母）来区分, e.g. `shop` (实心)和 `shop-o` (描线)。
- `type` 的命名规则: `icon名字-[variant(变体)]-[描线与否(o)]`。

### API

| 参数      | 说明           | 类型    | 默认值  |
| --------- | -------------- | ------- | ------- |
| type      | 图标类型       | string  | `''`    |
| spin      | 旋转动画       | boolean | `false` |
| className | 自定义额外类名 | string  | `''`    |
