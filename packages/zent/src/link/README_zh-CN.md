## Link

超链接，是对原生 `a` 标签的一个样式封装，同时支持禁用链接。

### API

支持 `<a>` 的所有属性。

| 参数     | 说明     | 类型      | 是否必须 | 默认值  | 备选值 |
| -------- | -------- | --------- | -------- | ------- | ------ |
| disabled | 是否禁用 | `boolean` | 否       | `false` | `true` |
