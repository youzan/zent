## Link

Link is a wrapper around native `a` tag, with an option to disable it.

### API

Supports all props in `<a>`.

| Property | Description      | Type      | Required | Default | Alternative |
| -------- | ---------------- | --------- | -------- | ------- | ----------- |
| disabled | Link is disabled | `boolean` | No       | `false` | `true`      |
