## Icon 图标

语义化的图标库。

### 使用指南

-   部分图标分为实心和描线两个版本, 使用 `-o`（字母）来区分, e.g. `shop` (实心)和 `shop-o` (描线).
-   命名规则: `icon名字-[variant(变体)]-[描线与否(o)]`.

### 代码演示

:::demo React组件方式使用
```js
import { Icon } from 'zent';

ReactDOM.render(
	<Icon type="youzan" />
	, mountNode
);
```
:::

:::demo 只使用CSS
```js
ReactDOM.render(
	<i className="zenticon zenticon-youzan"></i>
	, mountNode
);
```
:::
